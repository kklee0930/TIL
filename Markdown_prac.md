## Code Block(코드 블럭)

- backtick 기호를 3개를 활용하여 작성한다.
- 특정 언어를 명시하면 Syntax Highlighting 기능이 적용된다.

``` python	
print("Hello")

if True:
    print("t")
if False:
    print("f")
#주석
```
### Inline Code Block

`print`는 파이썬에서 출력하는 함수이다.
